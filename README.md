# Smart Complaint Management System

A Java + MySQL based Complaint Management System developed using Java Swing and JDBC.

## Features

* Submit complaints
* Automatic priority detection
* View all complaints
* Search complaints
* Resolve complaints
* Complaint statistics

## Technologies Used

* Java
* Java Swing
* MySQL
* JDBC

## Database

Database Name:
`complaints_db`

Table:
`complaints`

## How to Run

1. Install Java and MySQL
2. Create database and table
3. Add MySQL JDBC Connector
4. Run `Main.java`
